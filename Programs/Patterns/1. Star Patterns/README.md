# Star Patterns

## Basic Star Patterns

### 1. Square pattern

```
* * * * *
* * * * *
* * * * *
* * * * *
* * * * *
```

### 2. Right-angled triangle (left-aligned)

```
*
**
***
****
*****
```

### 3. Right-angled triangle (right-aligned)

```
    *
   **
  ***
 ****
*****
```

### 4. Inverted right-angled triangle

```
*****
****
***
**
*
```

### 5. Pyramid

```
    *
   ***
  *****
 *******
*********
```

### 6. Inverted pyramid

```
*********
 *******
  *****
   ***
    *
```

### 7. Diamond

```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

### 8. Sandglass

```
*********
 *******
  *****
   ***
    *
   ***
  *****
 *******
*********
```

